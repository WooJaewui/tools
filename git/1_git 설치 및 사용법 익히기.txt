
 Git 설치 및 사용법 익히기.



 # 저장소 만들기. ( Github ) 

저장소의 필요성.
Git은 버전관리를 해주는 프로그램인데 이러한 버전관리를 저장할 데이터 저장소가 필요하다.
저장소는 Github를 많이 사용한다.

속성.
public : 모든 소스를 공개함.
private : 소스를 비공개.
Add .gitignore : 라이센스 등의 이유로 올리지 못하는 파일을 설정한다.

git 설치하기.
git-scm.com -> win 파일 다운로드 -> next만 누르고 설치.



 # git 사용하기.

버전확인.
git --version.

깃 설정.
아이디 설정 : git config --global user.name 깃허브아이디.
이메일 설정 : git config --global user.email 이메일주소.

깃 다운받기.
git clone 경로 : 내가 설정한 깃허브 아이디의 파일 다운받기.

git 간단한 기능.
git add 파일명 : path에 있는 파일을 repository에 추가한다.
git commit -m "커밋 메시지" : 커밋을 통해 repository에 추가한 파일을 1차 저장한다.
git push : commit한 데이터( 파일 )을 github repository 저장한다.
