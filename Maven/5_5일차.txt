 # 라이브러리 인덱싱 검색.

Maven 창.
Window -> show view -> Maven -> Maven Repositories.

Global Repositories 설정.
Global Repositories -> central -> Rebuild Index.

pom.xml 라이브러리 설정.
pom.xml에서 창 아래에 Dependencies 클릭.
Add에서 원하는 라이브러리 검색. ( Global Repositories가 Rebuild index 되어 있어야 사용 가능 )



 # 내가 만든 라이브러리 설치하기.

project install. ( jar파일 만들기 )
export -> Java -> JAR file.
run as -> Maven Install. ( pom.xml에서 jar로 설정되어 있을 경우 사용 )

내가만든 jar파일 라이브러리로 사용.
Maven Repositories에서 Local Respository Rebuild Index해서 파일 로드.
Dependencies에서 클래스이름을 쳐서 jar파일을 add해줌.

팁.
프로젝트에 포함되어 있는 라이브러리는 아이콘이 폴더형태로 보임.
install은 프로젝트를 Local Repository에 옮김.
deploy는 프로젝트를 Global Repository( central )에 옮김.
